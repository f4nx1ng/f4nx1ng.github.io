VScode用作远控手段

发表于 2024-10-08 分类于 奇技淫巧 阅读次数：

Visual Studio Code Remote - Tunnel 扩展允许您通过安全隧道连接到远程计算机，例如台式电脑或虚拟机 (VM)。您可以从任何地方的 VS Code 客户端连接到该计算机，无需 SSH。

VScode用作C2

开启方式code tunnel

特点：

1.大厂证书，回连域名和 IP 为微软 VS Code，正常开发人员也会使用，可以说是白域名和白 IP 2.HTTPS 加密，执行的命令和回显的数据均为 TLS 加密 3.操作简便

唯一缺点：

需要目标受害机器可以访问外网（此处外网指的是VScode官网），所以可能需要科学上网，或者是本来就已经挂载系统代理的机器做受控端。

如何使用

首先开启code tunnel

他会给一个网址，然后让我们登陆这个网址去输入验证码

授权成功之后，受控端的控制台上会返回一串url，这个url就是用于控制端实现远程控制的链接。

这里其实会有一个bug，在浏览器上的左下角可能有的时候并没有显示连接成功的字样但是事实上已经连接成功了。在连接之后就可以建一个终端执行任意命令，并且可以看到文件目录。

其关键就是要看目标机器的Server是否已经启动，如下图所示就是已经启动成功的样子。
